# JoyViewer - Webtoon & Novel Reader v4.4

JoyViewer는 로컬 환경에 저장된 웹툰 이미지, 압축 파일(.zip, .cbz), 그리고 텍스트 파일(.txt) 형식의 웹소설을 최적의 환경에서 감상할 수 있도록 설계된 강력한 윈도우용 웹툰 및 소설 통합 뷰어입니다. 직관적이고 미려한 UI와 더불어 완전히 자동화된 스크롤링 기능을 제공하여 쾌적한 정주행 환경을 제공합니다.

## 🌟 주요 기능 (Features)

### 1. 강력한 내 서재 (Library) 관리
- **다중 폴더 지원**: PC 여러 곳에 흩어져 있는 웹툰 폴더들을 '내 서재'에 모두 담아둘 수 있습니다.
- **자동 썸네일 생성**: 각 웹툰의 첫 번째 화에서 첫 번째 이미지를 실시간으로 추출하여 직관적인 2열 그리드(Grid) 형태의 썸네일 카드로 보여줍니다.
- **설정 자동 저장**: 추가해 둔 폴더 목록은 `joyviewer_config.json`에 안전하게 저장되어 뷰어를 껐다 켜도 그대로 유지됩니다.

### 2. 고급 오토스크롤 (Auto-Scroll)
손가락 하나 까딱하지 않고 감상할 수 있도록 고도화된 두 가지 스크롤 모드를 제공합니다.
- **부드러운 스크롤**: 물 흐르듯 일정한 속도(1~100단계 조절 가능)로 화면이 계속해서 내려갑니다.
- **끊어 읽기 (사람처럼)**: 실제 사람이 마우스 휠을 굴리듯 한 번 부드럽게 감속하며 이동(`px` 지정)하고, 지정된 시간(`초`)만큼 멈춰서 읽을 시간을 주는 지능적인 스크롤입니다.
- **다음 화 자동 넘김**: 스크롤이 마지막 페이지에 도달하면 끊어 읽기에 설정된 대기 시간만큼 멈춘 후, 클릭 없이 즉시 다음 화를 불러오고 스크롤을 이어서 시작합니다.

### 3. 직관적인 플레이어 컨트롤 & 단축키
유튜브나 넷플릭스처럼 미디어를 제어하듯 웹툰을 제어할 수 있습니다.
- **마우스 컨트롤**: 상단 바의 재생/일시정지(▶/⏸) 토글 버튼과 정지(⏹) 버튼으로 직관적 제어.
- **화면 가장자리 이동 버튼**: 화면 양옆의 반투명한 `<` `>` 버튼을 클릭해 손쉽게 이전/다음 화로 이동합니다.
- **키보드 단축키**:
  - `스페이스바 (Space)` : 오토스크롤 재생 및 일시정지
  - `위/아래 방향키 (↑/↓)` : 오토스크롤 중에도 언제든 수동으로 위아래로 빠르게 이동
  - `좌/우 방향키 (←/→)` : 즉시 이전 화 또는 다음 화로 이동

### 4. 압도적인 성능과 동적 스트리밍
- **무압축 실시간 로딩**: 수백 MB의 무거운 압축 파일(`.zip`, `.cbz`)을 임시 폴더에 푸는 과정 없이 메모리에서 즉시 렌더링합니다. 따라서 아무리 용량이 커도 클릭 즉시 첫 페이지를 보여줍니다.
- **멀티스레드(Multi-thread) 이미지 서빙**: 내장된 Bottle 웹 서버가 멀티스레딩으로 동작하여 여러 장의 이미지를 병목 현상 없이 동시에 부드럽게 불러옵니다.

### 5. 모바일/스마트기기 스트리밍 서버 지원 (v3.7 업데이트)
PC에 켜둔 JoyViewer가 그 자체로 강력한 **웹툰 스트리밍 서버** 역할을 합니다. 
- **앱 설치 불필요**: 같은 Wi-Fi 환경에 있는 스마트폰, 태블릿 등의 웹 브라우저 주소창에 PC 뷰어 상단에 표시된 주소(예: `http://192.168.X.X:58210`)를 입력하기만 하면 즉시 접속됩니다.
- **모바일 최적화 몰입형 UI (Immersive)**: 스마트폰의 좁은 화면을 최대한 활용하기 위해 스마트 자동 숨김 기능이 적용되었습니다. 
  - 오토스크롤을 재생(▶)하면 상단 메뉴가 완전히 숨겨지며 화면 전체로 쾌적하게 감상할 수 있습니다. 
  - 수동으로 읽기 위해 정지(⏹) 시, 좁은 화면을 방해하지 않도록 메뉴 대신 작고 심플한 **A(Auto)** 및 **I(Index, 책갈피)** 버튼만 미니멀하게 제공됩니다.
  - 화면 우측 하단의 **전체화면(⛶) 버튼**을 누르면 브라우저의 주소창은 물론 스마트폰 하단 홈 버튼(내비게이션 바)까지 완벽하게 숨겨집니다.
  - 화면 가장자리의 이전/다음 화 넘기기 버튼 크기를 대폭 줄이고 벽에 완전히 밀착시켜 스와이프 및 터치 간섭을 최소화했습니다.
- **원격 접속 보안 (Security)**: 외부에서 접속할 때 보안을 위해 암호를 설정할 수 있습니다. 
  - PC용 앱 사이드바에서 현재 서버 주소 확인 및 접속 암호 설정이 가능합니다. 
  - 암호 설정 시 외부 기기(스마트폰 등)에서 접속할 때 로그인 화면이 나타나며 인증 없이는 감상이 불가능합니다.
- **용량 최적화**: 수십 GB에 달하는 웹툰을 모바일 기기로 복사할 필요 없이, 침대에 누워 편안하게 실시간으로 스트리밍 감상이 가능합니다.

### 6. 웹소설(텍스트) 감상 완벽 지원 및 맞춤형 설정 (v4.4 업데이트)
- **일반 텍스트(.txt) 및 압축 텍스트(.zip, .cbz) 지원**: 개별 `.txt` 소설 파일은 물론, 여러 텍스트 파일이 들어있는 `.zip` / `.cbz` 압축 소설 파일도 자동으로 압축 해제 없이 실시간 병합 정렬하여 감상할 수 있습니다.
- **글꼴 종류 및 크기 맞춤 설정**: 독자의 눈에 가장 적절하고 가독성이 뛰어난 글꼴 크기(14px ~ 36px)와 선호하는 서체(Noto Serif, Noto Sans, 나눔명조, 나눔고딕, system default 등 6개 옵션)를 화면 우측 상단의 `설정(⚙️)` 메뉴에서 실시간으로 조절하고 저장할 수 있습니다.
- **오토스크롤 적용 가능**: 웹툰과 마찬가지로 소설 뷰어 상태에서도 '부드러운 스크롤' 및 '끊어 읽기' 오토스크롤 방식을 그대로 적용하여 한 편의 소설을 끊김 없이 감상할 수 있습니다.

## 🚀 실행 방법

### 방법 1. 간편 실행 (권장)
폴더에 함께 제공되는 `run_viewer.bat` 파일을 더블 클릭합니다.
- 파이썬 환경을 체크하고, 필요한 라이브러리가 없다면 자동으로 설치한 뒤 뷰어를 띄워줍니다.

### 방법 2. 수동 실행
터미널이나 명령 프롬프트에서 아래 명령어를 입력합니다.
```bash
pip install -r requirements.txt
python webtoon_viewer.py
```

## ⚙️ 요구 사양 및 종속성
- Python 3.8 이상 권장
- `pywebview`: 앱을 윈도우 창으로 띄우는 GUI 프레임워크
- `bottle`: 로컬 파일들을 빠르고 안전하게 제공하는 경량 웹 서버
- `Pillow`: 고급 이미지 처리 및 썸네일 분석

## 💡 개발 팁 (For Developers)
- `JoyViewer`는 모던 웹 기술(HTML, CSS, JS)과 파이썬 백엔드가 결합된 하이브리드 앱입니다. 
- UI나 테마(색상)를 변경하고 싶다면 `webtoon_viewer.py` 파일 내의 `HTML_CONTENT` 블록에 위치한 CSS/JS 코드를 직접 수정하여 마음껏 커스터마이징할 수 있습니다.
- 모든 스크롤 및 지연 로딩 설정값들은 앱 재시작 시 `joyviewer_config.json`에서 자동으로 로드됩니다.
